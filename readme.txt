CartFlows is a sales funnel builder that allows you to transform your WordPress website into a sales platform. Avoid confusing third-party messaging with Rebrand CartFlows so your logo and colors are front and center. Rename the plugin and change the description, and also easily replace the developer’s name with your company’s

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress(This will add new Sub Menu in admin panel's side menu name Rebrand` under CartFlows menu).
3. Go to CartFlows > Rebrand

== Frequently Asked Questions ==
1. Is the Rebrand CartFlows plugin free? 
-Yes, we have a lite version of Rebrand CartFlows plugin with limited features. If you want to take advantage of all the premium features you'll need to purchase a Pro license.

2. Can you use the Rebrand CartFlows plugin on the free version of CartFlows? 
-Yes, it will work with both free version and pro version.

3. Does it work with any WordPress theme? 
-Yes, it will work with any standard WordPress theme.

4. Does it work with any WordPress page builder? 
-Yes, it will work with any WordPress page builder.

5. How many websites can I connect Rebrand CartFlows to? 
-You can connect the lite version to as many websites you would like. If you want to use the Pro features, you'll have to select the plan that fits your needs best.
